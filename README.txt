Plugin to add, edit all posts and pages from different websites on 1 website.

== Description ==

This is a plugin to manage all your posts and pages from all websites from 1 website.

The only thing that u have to do is add the plugin code that u can find on the wordpress website on the management website (https://blog.tool.wux.nl/).
On this website u can manage everything.

== Installation ==

Simply install as a normal WordPress plugin and activate.

Make an account at (https://blog.tool.wux.nl/).
Make a company.
Add your website.
